# Style Guide

NB: Note capitalization throughout.

## Technical Terms

online
: **not:** "on-line"

The World-Wide Web:
: "The Web" (for short)
: adjective: online (e.g, "If the item is not itself available **online**, but")
: **not:** "WWW"
